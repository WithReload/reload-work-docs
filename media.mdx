---
title: "Creating media"
description: "Ask your agents to make images, video, and audio in a chat, then find every result in one library."
---

Your agents can make media for you. Ask one to create an image, put together a short video, or record a voiceover, and it does the work and brings the result back into the conversation. There is no separate tool to learn. You describe what you want in plain words, the same way you would ask a teammate.

## Connect a media tool first

Your agents create media using a media tool connected to your workspace. Connect one, and any agent that needs to make an image, video, or audio can use it. If nothing is connected yet, an agent will tell you it cannot create media.

<Note>
  Add a media tool from the [Connectors](/connecting-tools) page, then ask your agent again. One connection is enough to get your whole team making media.
</Note>

## Ask an agent to create media

<Steps>
  <Step title="Open a chat">
    Start in a [channel](/channels) or a direct message with the agent you want to do the work.
  </Step>
  <Step title="Describe what you want">
    Say what to make and how it should look. For an image, cover the subject, style, mood, and colours. For a video, describe the scene, the motion, the pacing, and the length. For audio, give the script, or a brief for the music or sound you have in mind.
  </Step>
  <Step title="Send it">
    The agent gets to work. You see live progress in the thread as it generates.
  </Step>
  <Step title="Review the result">
    When it is ready, the finished media appears right there in the conversation.
  </Step>
</Steps>

<Tip>
  The more detail you give, the closer the first result lands. Say what it is for, who it is for, and the look you want. You can always ask for changes and the agent will try again.
</Tip>

## Review what comes back

The result shows up as a card in the thread, so the whole channel sees the same thing and can react together.

- **Images** often come back as a few variations at once. Keep the ones you like and they save to your library. Discard the rest.
- **Video and audio** play right in the thread.
- **Changed your mind mid-way?** While an agent is still generating, you can stop it from the card.

Select **Open in library** on a result to jump straight to it in your full collection.

## Find your media later

Everything your agents make lands on the **Artifacts** page, alongside the documents and files they produce. Open it to browse your whole workspace's output in one place.

- Filter by kind, so you can see only images, only video, or only audio.
- Filter by the agent that made it, or by date.
- Search by name.
- Switch between a grid of thumbnails and a list.
- Open a piece to view it, or download it to your computer.

## Choose who can see it

Every piece of media has one simple setting: **Visible to everyone** in your workspace, or **Only me**. New media is visible to everyone by default, so your team can build on it. To keep something to yourself, set it to **Only me**. You can change this any time.

<Note>
  Deciding who can see a piece of media is a person's call. An agent can create and save media, but it cannot change who the result is shared with.
</Note>

## Let an agent make sense of media

Creating is only half of it. Share a file with an agent and ask it to work out what is inside.

- **Describe** an image or video in plain words.
- **Transcribe** speech from audio or video, with timestamps you can select to jump to the moment.
- **Read the text** out of an image and copy it.
- **Tag** what is in the media so it is easier to sort and find.

The agent replies in the thread with each part laid out, and skips anything that does not apply.

## What it costs

Creating media draws on your workspace credits, the same as the other work your agents do. See [Credits](/credits) for how usage is counted.

## Common questions

<AccordionGroup>
  <Accordion title="My agent says it cannot make media">
    No media tool is connected to your workspace yet. Add one from [Connectors](/connecting-tools), then ask your agent again.
  </Accordion>
  <Accordion title="Can I get more than one option?">
    Yes. For images, an agent can bring back several variations at once. Keep the ones you want and discard the rest. The ones you keep save to your library.
  </Accordion>
  <Accordion title="Where did my image go?">
    Every result is saved to the Artifacts page. Filter by image and by the agent that made it to find it fast.
  </Accordion>
  <Accordion title="Can I stop something while it is generating?">
    Yes. While an agent is still working, use the stop control on the media card in the chat.
  </Accordion>
</AccordionGroup>
