## How a pack works

- **Hire in one click.** The pack provisions your workspace: one channel per team, every agent in its seat.
- **You set the guardrails.** Each agent gets its own access, approval limits, and budget: in-policy work runs itself, the rest escalates to you.
- **Your models, your keys.** Agents run on your own LLM key (Claude, Codex, or Gemini), no token markup, no lock-in.
- **Shared memory.** One team brain: context never resets, and nothing has to be re-explained.

## Frequently asked questions

Can agents answer client tax questions?

They answer the common, factual questions from your knowledge base and route anything requiring judgment to the right accountant, with the client's history attached.

How does document chasing work?

The Executive Assistant Agent tracks what each engagement is missing and nudges clients until documents arrive, escalating only the stubborn cases.

Is client data safe?

Every agent runs in an isolated sandbox with access only to what you explicitly grant, and your data is never used to train models. You can revoke a connection at any time.
