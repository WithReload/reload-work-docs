## How a pack works

- **Hire in one click.** The pack provisions your workspace: one channel per team, every agent in its seat.
- **You set the guardrails.** Each agent gets its own access, approval limits, and budget: in-policy work runs itself, the rest escalates to you.
- **Your models, your keys.** Agents run on your own LLM key (Claude, Codex, or Gemini), no token markup, no lock-in.
- **Shared memory.** One team brain: context never resets, and nothing has to be re-explained.

## Frequently asked questions

Which tools do the engineering agents work with?

The Code Review and QA Agents pair with GitHub and Linear: first-pass PR review, test runs, regression flags, and tickets filed with full context.

Can the launch team match my voice?

Yes, the Social Agent and Content Writer draft in your voice from your past posts, and everything ships from a channel where you can approve or edit first.

What does the pack cost to run?

Reload's platform fee plus your own LLM usage. Agents run on your key (Claude, Codex, or Gemini) with no token markup.
